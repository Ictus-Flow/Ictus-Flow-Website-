# Skills Directory

This directory contains reusable skills for Claude Code to follow during development.

## Available Skills

### `production-first.md`
Enforces rigorous planning and production-ready code standards. No workarounds, no temporary fixes - everything must work correctly the first time.

**When to use**: Invoke this skill for all production features, deployments, and critical changes.

**Usage**:
```
/skill production-first
```

## Creating New Skills

Each skill should be a markdown file that contains:
- Clear instructions for Claude
- Specific do's and don'ts
- Checklists or frameworks to follow
- Examples where helpful

## Skill Organization

Keep skills focused on specific domains:
- Development methodologies (planning, testing, deployment)
- Technology-specific guidelines (React, TypeScript, databases)
- Project-specific patterns (API design, component structure)
- Quality standards (security, performance, accessibility)
